# Minishell

A minimal UNIX shell implementation written in C

---

## Tests

This project includes a lexer test suite.

Build and run lexer tests:
``` 
make run-test-lexer

``` 

Run lexer tests with Valgrind:
``` 
make run-test-lexer-valgrind
``` 

Valgrind suppressions for readline are handled automatically if available.
